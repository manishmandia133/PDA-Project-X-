# Pushdown Automaton — Interactive Theory & Simulator

An interactive web-based platform for learning, visualizing, and simulating Pushdown Automata (PDAs). Understand context-free languages through hands-on experimentation with formal language theory.

## 🎯 About This Project

A **Pushdown Automaton** is a computational model that extends finite automata with a stack—giving it the power to recognize all **context-free languages** (Type 2 in the Chomsky hierarchy). This project provides:

- **Interactive Theory**: Deep dive into PDA formal definitions, transitions, and acceptance modes
- **Live Simulator**: Step through computations with real-time stack visualization
- **Worked Examples**: Explore classic CFLs like aⁿbⁿ, palindromes, and balanced brackets
- **Real-World Context**: See how PDAs power compilers, parsers, and language processors
