FORMAL DEFINITION

A Pushdown Automaton is defined as a 7-tuple M = (Q, Σ, Γ, δ, q₀, Z₀, F). It extends a finite automaton with a stack, giving it the power to recognize all context-free languages — exactly Type 2 in the Chomsky hierarchy.

SymbolNameDescription
QStatesFinite set of states the automaton can occupy
ΣInput AlphabetSymbols that can appear on the input tape
ΓStack AlphabetSymbols that can be pushed/popped from the stack; Γ ⊇ Σ typically
δTransition FunctionQ × (Σ∪{ε}) × Γ → 2^(Q×Γ*). Maps state + input + stack-top to a set of next configurations
q₀Start Stateq₀ ∈ Q. The computation begins here
Z₀Bottom MarkerZ₀ ∈ Γ. Placed at the bottom of the stack before any input is read
FAccept StatesF ⊆ Q. Used in acceptance by final state
TRANSITION NOTATION

A transition rule δ(q, a, A) = (p, γ) means: in state q, reading symbol 'a', with stack top 'A' — move to state p, pop A, and push γ. If γ = ε the result is a pure pop. If γ = AB then B is pushed first, then A — so A ends up on top.

δ(q, a, A) = (p, γ)
→ q current state
→ a input symbol consumed (ε = no consumption)
→ A stack top popped
→ p next state
→ γ string pushed to stack (ε = pop only)
Accept by Final State
Input consumed AND current state q ∈ F. Stack contents irrelevant. Most common in practice.
Accept by Empty Stack
Input consumed AND stack is empty. State irrelevant. Formally equivalent — any PDA can be converted between modes.
CHOMSKY HIERARCHY

PDAs correspond exactly to Type 2 — context-free grammars. They strictly subsume regular languages (DFA/NFA) and are strictly subsumed by Turing Machines.

PUMPING LEMMA

The Pumping Lemma for CFLs proves certain languages are NOT context-free. Classic non-example: L = {aⁿbⁿcⁿ}. A PDA cannot recognize it — it has one stack (one counter) and needs two independent counters.

For all CFL L, ∃ pumping length p:
Any s ∈ L with |s| ≥ p → s = uvxyz where:
① |vy| ≥ 1   ② |vxy| ≤ p   ③ uvⁱxyⁱz ∈ L for all i ≥ 0

If ANY string in a language fails this, the language is NOT a CFL.
